# vite-tool-template

## 如何使用模板？

```
// 安装依赖
yarn

// 测试
yarn test

// 唤醒测试ui界面
yarn test:ui

// 使用 istanbul 计算测试覆盖率
yarn coverage

// 构建
yarn build
```
